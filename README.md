# 📊 Employee Analytics Dashboard – Power BI

This repository presents an interactive Power BI dashboard developed to deliver meaningful insights into employee productivity, project distribution, and managerial performance across various departments.

---

## 📁 Overview

The Employee Analytics Dashboard serves as a business intelligence solution that highlights key workforce metrics, including:

- Employee billed hours
- Task categorization and time allocation
- Project-level resource utilization
- Department-wise workforce distribution
- Manager performance tracking

It enables stakeholders such as HR teams, project managers, and leadership to monitor performance indicators and make informed decisions regarding resource planning.

---

## 📌 Key Highlights

### Manager Insights
- Detailed view of billed hours attributed to each manager
- Comparative analysis of manager performance
- Contribution breakdown across departments

### Workforce Distribution
- Representation of workforce categories (Employees, Interns, Trainees)
- Team segmentation across:
  - BI
  - Development
  - Data Warehouse
  - HR
  - Sales & Marketing

### Task-Based Time Analysis
- Distribution of hours across different activities:
  - Testing
  - Development
  - Customization
  - Reporting & Dashboard creation

### Project-Level Metrics
- Allocation of billed resources across projects
- Budget distribution across task categories
- Comparison of billable vs non-billable hours

### Time Trends
- Monthly and quarterly analysis of billed hours
- Year-over-year comparison (2015–2017)

---

## 🎯 Objective

This dashboard is designed to address key operational questions, such as:

- Which managers and departments demonstrate optimal efficiency?
- How is employee time distributed across tasks?
- Are projects aligned with budget and billing expectations?
- What trends can be observed across time and workforce roles?

---

## 🛠 Technology Stack

- Power BI Desktop
- Data transformation using ETL techniques
- Visual elements including:
  - Bar and column charts
  - Stacked visualizations
  - KPI indicators
  - Interactive filters and slicers
